# 🏫 College Campus VR Tour

A 3D virtual tour of a college SCIS Department at University Of Hyderabad campus built using A-Frame (WebVR).  
Users can explore the campus, walk through corridors, climb stairs, and take a guided tour.

## 🚀 Features
- First-person navigation (WASD + mouse)
- Auto-walk guided tour
- Realistic textures & lighting
- Boundary control (no wall crossing)

## ▶️ How to Run
Use a local server (Live Server in VS Code) and open in browser using:
http://localhost

## 🛠️ Tech Used
HTML, CSS, JavaScript, A-Frame
